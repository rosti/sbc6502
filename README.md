# sbc6502

Emulation for a very simple MOS Technology 6502 based Single Board Computer (SBC).

## Installation

Git clone (or download) from: https://github.com/rosti/sbc6502

## Usage

    $ lein run <intel-hex-rom1> [<intel-hex-rom2> ...]
